The VIVO Pump and Simple VIVO

The VIVO Pump is Python software for managing data in VIVO using spreadsheets. Simple VIVO is a command line tool to use the Pump to read spreadsheets and use them to update VIVO, or to get data from VIVO into spreadsheets for data management.

Scholarly Work

Simple VIVO provides "definitions" that convert your spreadsheet data to and from VIVO's internal semantic web formats. Using these maps, you can manage a wide variety of data useful for representing the scholarly work of your institution. Simple VIVO provides maps for people, their positions, educational background, awards and honors, publications, grants, teaching, mentoring, and service. No knowledge of the semantic web or the VIVO information representation formats is needed. It's simple.

Simple VIVO Data

Simple VIVO comes with data useful to representing the work of your organization. Using the provided data reduces the effort to manage data in your VIVO.

The Simple VIVO Book

Simple VIVO and the VIVO Pump are described in an e-book which can be read on-line, or downloaded as a PDF, or for iBook, or Kindle. The book describes Simple VIVO configuration, examples suitable for managing scholarly data using VIVO, and reference materials for both Simple VIVO and the Pump.
