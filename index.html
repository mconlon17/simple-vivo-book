The VIVO Pump and Simple VIVO

The VIVO Pump is Python software for managing data in VIVO using spreadsheets. Simple VIVO is a command line tool to use the Pump to read spreadsheets and use them to update VIVO, or to get data from VIVO into spreadsheets for data management.

The Simple VIVO Book

Simple VIVO and the VIVO Pump are described in an e-book which can be read on-line, or downloaded as a PDF, or for iBook, or Kindle. The book describes Simple VIVO configuration, examples suitable for managing scholarly data using VIVO, and reference materials for both Simple VIVO and the Pump.
